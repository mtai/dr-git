========================================
Installation
========================================
1) cd <INSTALL_DIR>

2) git clone https://github.com/mtai/dr-git.git

3) vi ~/.bashrc
    DR_GIT_HOME=<INSTALL_DIR>/dr-git/
    PATH=$PATH:${DR_GIT_HOME}/bin/

4) source ~/.bashrc

5) To test, go to any git dir and run "git dr-init"
